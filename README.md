# Tetris2
Five people online ， tetris


#2006年写的 6人联机 游戏 《火拼俄罗斯》，使用C#制作， 很多年前 传到 csdn 上了。 
注： 主要技术点
  １,　设计模块 ( bridge , instance 等 )
  2，  双缓冲绘制 ( 防止界面闪烁 常用  )
  3，  数据结构 （ 矩阵转置 ）
  4，  消息的异步通知  （主要是网络消息的异步通知）
  5 ,  游戏界面信息的差异传输已经数据编码  
  6 ,  tcp 的网络编程
